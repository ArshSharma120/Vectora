## ğŸ› ï¸ Installation

### 1. Load Extension in Chrome

1. Download or clone the `extension` folder
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the `extension` folder
6. The Vectora icon should appear in your toolbar!

### 2. Configure API Keys

1. Right-click the Vectora extension icon
2. Click **Options** (or go to extension settings)
3. Select your preferred AI provider:
   - **Cerebras** - Fast inference with Llama models
   - **Gemini** - Google's multimodal AI (supports text & images)
   - **Groq** - Ultra-fast inference (supports vision models)

4. Enter your API key:
   - **Cerebras**: Get from https://cloud.cerebras.ai/
   - **Gemini**: Get from https://aistudio.google.com/apikey
   - **Groq**: Get from https://console.groq.com/keys

5. Select your preferred model
6. Click **Test Connection** to verify
7. Click **Save Settings**
